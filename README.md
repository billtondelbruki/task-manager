# TaskFlow – To-Do List App

A simple, interactive task manager built with HTML, CSS, and JavaScript. Focuses on DOM manipulation, state management, and localStorage persistence.

---

## Core Features

- Add tasks (button or Enter key)
- Mark complete/incomplete (click checkbox or text)
- Delete individual tasks
- Filter: All / Active / Completed
- Clear all completed tasks at once
- Tasks persist after page refresh (localStorage)

---

## Visual Highlights

- Centered layout with max width
- Completed tasks: strikethrough + lighter color
- Empty state message (e.g., "No tasks for today!")
- Gradient background & hover effects
- Fully responsive (mobile + desktop)
